This is an example document that opens in a window when you click on it from My Documents!

Features

  • Opens in a draggable, resizable window
  • Can be minimized, maximized, or closed
  • Displays any HTML content you want
  • Works just like other Windows 2000 windows
Note: You can create any HTML file and have it open this way. Just add it to the file list in My Documents with the appropriate function call!

How It Works

When you double-click a file in My Documents, it calls a function in the parent window that creates a new window with an iframe containing your HTML file.

You can add any content you want to these documents - text, images, links, tables, forms, or anything else HTML supports!

Examples of What You Can Do

  • Create a personal resume or CV
  • Make a photo gallery
  • Build a project showcase
  • Write blog posts or articles
  • Create interactive forms
  • Display data visualizations

This is just an example - the possibilities are endless!
